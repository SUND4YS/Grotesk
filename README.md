# Grotesk — Тестовое задание (MODX 3)

Проект представляет собой реализацию верстки и интеграцию контента в CMS **MODX Revolution 3**. Сайт успешно развернут на VPS под управлением Ubuntu. на адресе (http://grotesk.mooo.com/)

## 🛠 Стек технологий
* **CMS:** MODX 3.0.5+
* **Frontend:** HTML5, CSS3, JavaScript
* **Environment:** VPS (Linux/Ubuntu), LAMP/LEMP Stack
* **Version Control:** Git

## 📋 Особенности реализации
* **Миграция:** Проект успешно перенесен с локальной среды разработки (XAMPP/Windows) на VPS. Проведена полная настройка путей и конфигурации Namespaces.
* **Структура:** 
  - Основные ресурсы и шаблоны настроены в админ-панели MODX.
  - Все статические файлы (CSS, JS, Fonts, Images) вынесены в директорию `/assets/`.
* **SEO & UX:** 
  - Настроены дружественные URL (FURL).
